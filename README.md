# Exercícios da faculdade de Ciência da Computação

Este repositório serve como um portfólio para os projetos e exercícios que estou desenvolvendo durante a minha graduação em Bacharelado em Ciência da Computação, no IFSP Campus Salto. A função do repositório é demonstrar os meus conhecimentos obtidos durante as aulas da faculdade, desde as primeiras aulas até as últimas.


## Estrutura do Repositório
O conteúdo está organizado por disciplina, com cada pasta contendo os projetos e as atividades específicas. Cada pasta de disciplina pode ter sua própria estrutura, dependendo dos projetos, mas a ideia é manter tudo o mais organizado possível.


## Tecnologias e Linguagens Utilizadas
Abaixo está uma lista das principais tecnologias e linguagens que estou utilizando nos projetos. Atualizarei conforme aprender novas linguagens na faculdade. **Não implica que eu não conheça outras lingugaens**, mas que apenas estas linguagens foram utilizadas nas aulas da minha graduação. 

* Linguagens: C

* Ferramentas: Git, GitHub, VSCode.

* Frameworks: Ainda não utilizei

* Bibliotecas: Ainda não utilizei


## Como Navegar
Para explorar os exercícios e projetos, basta navegar pelas pastas de cada disciplina. Dentro das pastas das matérias haverá um README.md com informações detalhadas sobre os exercícios ou projeto em específico.
